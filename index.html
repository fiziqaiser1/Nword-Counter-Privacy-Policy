Anzu Bot - Privacy Policy

By using Anzu discord bot, you accept its privacy policy.

1. What does it store?

Server IDs: for server data (welcome messages).

Channel IDs: for sending the welcome message when a member joins a server in the channel provided by the user.

Welcome Message content: for the welcome message to display.

Welcome Image: for the background image of the welcome message.

2. Who can access the data?

2.1 Following people can access ALL of the data:

  • Bot developer(s)

2.2 Following people can access reports:

  • Bot developer(s)

3. How can I get rid of the data stored?

3.1 To remove the server id or channel id stored in our database do one of the following:

  • Contact any of the server moderators in our discord support server.
  • Kick the bot from the server.

4. Why does Anzu store this data?

4.1 Anzu stores server IDs for settings and data because:

  • Without those, most of the bot wouldn't be functional.

4.2 Anzu stores channel IDs for sending the welcome message to the channel provided by the user:

  • Anzu needs to send the welcome message to a dedicated channel. The user will provide the channel ID where the welcome message will be sent.

4.3 Anzu stores Welcome Message Content provided by the user to display a message when a new member joins the server:

  • This data is not necessary for Anzu's functionality. This just gives a user a nice option for customizing and adding aesthetics to the discord server.

4.4 Anzu stores Welcome Image provided by the user to display a image when a new member joins the server:

  • This data is not necessary for Anzu's functionality. This just gives a user a nice option for customizing and adding aesthetics to the discord server.

5. How do I get all my data deleted?

Simply join our support server to request deletion of your data, Your entire data will be then be deleted within 24 hours or just kick the bot from the server.

6. Why do you need all this data, this Privacy Policy doesn't answer my question. What should I do?

You can simply join our discord support server and we can have a chat over a nice coffee :)
