N-Word Counter Bot - Privacy Policy

By using N-Word Counter discord bot, you accept its privacy policy.

1. What does it store?

Server IDs: for server data.

User IDs: To keep track each users N-Word count.

N-Word Count: The count for the user typing words beginning with 'N' or 'n'.

2. Who can access the data?

2.1 Following people can access ALL of the data:

  • Bot developer(s)

2.2 Following people can access reports:

  • Bot developer(s)

3. How can I get rid of the data stored?

3.1 To remove the server id or user id stored in our database do one of the following:

  • Contact any of the server moderators in our discord support server.
  • The bot's data is deleted on the 1st of every month to reset the stats.

4. Why does N-Word Counter store this data?

4.1 N-Word Counter stores server IDs for settings and data because:

  • Without those, most of the bot wouldn't be functional.

4.2 N-Word Counter stores User IDs for keeping track of the user typed the words start with 'N' or 'n':

  • This will help the bot to display the stats of users word counts.

4.3 N-Word Counter stores the count of words beginning with 'N' or 'n' typed by the user:

  • This data is for N-Word Counter's functionality.

5. How do I get all my data deleted?

Simply join our support server to request deletion of your data, Your entire data will be then be deleted within 24 hours.

6. Why do you need all this data, this Privacy Policy doesn't answer my question. What should I do?

You can simply join our discord support server and we can have a chat over a nice coffee :)
